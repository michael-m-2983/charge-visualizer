# Charge Visualizer

This is a simple web app that visualizes the electrical potential around some charges. There is an experimental feature for showing equipotential lines as well, but it can only show a single line at a time for the moment.

## Usage

Visit the live demo hosted at <https://michael-m-2983.github.io/charge-visualizer>.
